Where there is a tree, there is a country

发表于 2018-03-16 分类于 UK&Ireland ， ScotEire Valine：
3.5k 9 分钟

　　新乌尔斯特命运多舛。乘着帆船渡过大西洋的爱尔兰因为从荷兰人的嘴边抢走新生幼童而欢欣鼓舞，随即陷入近千年来逃不掉的教育怪圈。黄金时代结束之后，他在饱含大麻气味的广袤土地中睁开眼，知晓自己与圣徒岛血脉相连，但依旧充满对独立个体的内层次需求。他搞不懂自己为何诞生，看到年幼的阿尔弗雷德便自觉与他不同，尽管那时候他并没有足够的思想来解释为什么不同。他站在与自己面目相似的爱尔兰面前：“我想做个与你一样的独立国家。”

　　爱尔兰认为他很有想法。他获得了阶段性胜利，代价是被荷兰和西班牙揍得像只兔子，对着镜子处理脸上的伤口，听他这么说，想给他一个鼓励的微笑，没成功，捂着自己的脸说不出话，疼得眼泪差点下来。等到他终于缓过劲来，对他比手势：“想法很好。但是我也帮不了你。你知道我现在朝不保夕，说不定哪一天一睁眼，我就成了另外某种存在的一部分。我怎么教你们的？风起云涌，瞬息万变，简直媲美原始森林，稍有不慎，骨头都被吃掉。”

　　“不过你可以退而求其次，”爱尔兰说，“你可以做个独立的人。在这个世界上，保持独立和清醒的人格也是一种伟大的成就。”

　　于是新乌尔斯特决定做个独立的人。这个想法一直在他的脑子里盘旋，随后进入他的血液，再到后来成为组成他身体的每一部分。他活着，像凭空出现的奇迹一般活着，活到吞掉罗德岛，活到新泽西划进版图，活到冲突结束进入联盟，活到重新被分成六块。他再见到阿尔弗雷德，现在叫美利坚，美利坚年轻、活跃、怀抱合理或无来由乐观的大男孩并不遮掩自己的诧异：“老兄！天哪，你怎么还活着？”

　　“按理说你现在应该成为我的心脏，脾胃，或者别的什么，”美利坚手舞足蹈，“可你看起来还是个人！既然你看起来还是个人，那你是什么？”

　　新乌尔斯特现在已经可以思考。但思考不一定就能给予他结果。他只能说出最初的那个答案：“我是个人。”

　　美利坚很给他面子：“太酷了吧！”

　　　　　　　

　　所以新乌尔斯特并未成为他的一部分，他倒是不觉得哪里酷了，美利坚的想法总是很难揣摩。美利坚已经不去做礼拜了，他仍然还去，例行公事，保留着血脉里那份浓厚的宗教基因。他像个普通人一般活着，偶尔行驶应当拥有的部分权力，地位奇特，处境尴尬，被卡在半空，不上不下。后来他检讨自己的半生，终于找到存在的发源：爱尔兰对他说，“你可以做个独立的人”。他照做了，所以多一份自我。倒是爱尔兰的乌鸦嘴事事灵验，他本人也不得不退而求其次，创造所谓的伟大成就；他也一样忘记了新乌尔斯特存在的理由。

　　但新乌尔斯特仍旧胸怀（不切实际的）爱、（实属渺茫的）希望与（永远不会从中取得教训的）记忆。战后二十多年，一切都趋于缓和，他踏上作为他源头的岛屿，看到爱尔兰举着自己的吊瓶，再次踏入那个逃不掉的教育怪圈。他对面的北英格兰用从搅拌机里倒出来的语言对他坦诚：“我想做个独立国家。”

　　爱尔兰扬起眉毛。“我怎么觉得听起来这么耳熟。”

　　“我也说过。”新乌尔斯特说，“大概三百年前说的。”

　　“这不一样。”爱尔兰说，“这话现在再说就是不合时宜。连扭曲的婚姻经过战争炮火洗礼后仍旧要苟延残喘，他却对我说他现在要做一个独立个体。现在这个时代，你哪怕逃到南太平洋做个孤岛，都没办法保证自己是独立个体。话说回来，你看我，我还是座岛呢。”

　　他换了只手拿药瓶。新乌尔斯特觉得这违反医学法则，但是关于这个他没说一个词。在听到他建议可以做独立的人后，爱尔兰继续他的演讲：“我是什么都无济于事。海作为我的天然屏障，却又不情愿如此，找到机会就背信弃义，而到头来我仍旧不得不信它。我们向人类靠拢，但人类也不一定每时每刻都能签订不需要感情的契约。所以我鼓励你，但我们仍然无法做人。尤其是北英格兰——”他转过脸，“听起来残酷，但我仍然要说。只有你最不可能做人。很不幸地，你是遗留问题的产物，是未被预料的夹缝，是拼凑起来的弗兰肯斯坦。”

　　新乌尔斯特为他感到难过。但是北英格兰并不难过。北英格兰说：“你这是诡辩，我不想做人。”

　　爱尔兰重新靠回椅背。他说气话：“我现在就要把利物浦的所有苏格兰人都赶走。”

　　“我爱你，”北英格兰用爱尔兰语说，“说真的，你才不知道我有多爱你。但我恨你肯定比爱你要深（英语），你才不知道我有多恨你。我不做人，人的恨与爱才不能并驾齐驱。我爱你的部分让我为你做了早餐，我恨你的部分让我给你的早餐茶中下了毒。”

　　“……操。”爱尔兰冷静地说，“还真他妈是你干的啊。”

　　新乌尔斯特知道他为什么在挂吊瓶了。他日后知道爱尔兰因为氰化物呼吸骤停了五分钟，而苦杏仁味被更浓厚的爱尔兰早餐茶的味道盖掉了。爱尔兰对他描述：“感觉也挺好的。你躺在那里，忘记呼吸，或者说不需要呼吸的拖累，就只是躺着而已。你躺着，什么都不想，眼前五光十色，就像身处群星之间。你看着繁星，繁星看着你，伸手就能触碰，尽管你抬不起手。”

　　（新乌尔斯特诧异他还是有如此传统的审美观。）

　　北英格兰用苏格兰通用语继续说：“我想做个独立国家。不像你，像他。但我仍然只能从你这里获取经验，因为斯科蒂才不会想看一个新的独立国家。”

　　爱尔兰说：“那你种棵树吧。”

　　　　　　　

　　身处枪林弹雨之中，爱尔兰也非常善于忙里偷闲。他在地图上的爱尔兰中央画了棵树。他自鸣得意，对苏格兰讲：我要是德国人，我就往这里轰炸。从人类形态的角度来讲，这里是心脏，我们杀人，除了头部之外，最常用的方法就是破坏心脏。从国家的角度来讲，这里是棵树……什么树？什么树都行，就当是棵橡树。你我都知道树是一个国家的本源……你别他妈插嘴。他扶正眼镜，继续说：有树的地方就有国家。你看吧，我宁可只有棵树。

　　苏格兰说：这样吧。你写篇论文，能论证国家和树之间要是存在客观联系，我给你拨款。

　　爱尔兰笑得伤口迸裂，流出红色的树汁来。

　　新乌尔斯特在爱尔兰这边住了一个星期。北英格兰不停用传真发来照片：一棵，两棵，三棵。爱尔兰心安理得地在扶手椅中生长，吃苏格兰给他削过皮还切了块的苹果。爱尔兰说：“你看他就不行。我只让他种一棵树，他种这么多棵。过几天还会继续给我发传真，直到种出一片森林他才罢休。”

　　“所以他成不了独立国家？”新乌尔斯特说。他提出的是非常合理的猜测，然而很快就被否决。

　　“不是，”爱尔兰说，“他成不了人。虽说成为独立国家的可能性微乎其微，但他可以做个不独立的国家。但他成不了人。人的胃口才没有这么大。”

　　“你看，”他又说，“我和苏格兰的差别就在这里。”

　　“可你说过我可以做个独立的人……你看北英格兰，信你也信得毫无保留。”

　　新乌尔斯特笑了起来。他不太关注爱尔兰与苏格兰之间的区别，他还是喜欢自己的那个成为人的愿望。现在某种意义上他得到了认可，因此觉得非常快乐。他说：“毕竟我们也可以签订有感情的契约。你可能会忽视这一点。”

　　他胸怀的爱仍旧是有来由的。稚拙，粗糙，没有结果，却仍旧有来由。血脉远比他想象的要影响深刻，他以前没什么感觉，现在却由衷地感谢血脉了。它像脐带一样，将他与母体连接起来，尽管他暴露在这个世界上已经许久，却仍然像是停留在温房。他仍然拥有那份模糊的原始记忆，那是他成为人的根本。

　　它的来由却貌似浑然不觉。爱尔兰看电视里放的二战纪录片，心血来潮，和苏格兰提起当时的事。爱尔兰始终认为自己对他的爱是少年时期的朦胧错觉，过了那个时间段，就很难称得上爱了。他痛斥苏格兰的野心，痛斥他想把整个世界吞掉的狂妄渴求，他巧言善辩，把自己从共犯的位置挪到了受害者一方。他宣布：自己是不得已。

　　“我没有你那样的胃口，”他说，“我其实只想做个独立的人。我只想做个人，坐在沙发上，看属于别人的历史。你想要的那些我从来都没有渴望过。”

　　“可你喜欢海，”苏格兰说，“你没法否认，你也喜欢拥有海。”

　　“我那是被你从陆地上逼下海，”爱尔兰反驳他，“海带给我的痛苦回忆远比快乐要多。你看，我就是你的流放者。我被流放到海的另一头，当然也不能全说是坏事。”

　　爱尔兰的自我矛盾第二天就奏效。他们前往码头，新乌尔斯特即将乘船返回新大陆。但爱尔兰，站在原地，睁大双眼，不可置信地，看着苏格兰为他展示一艘渔船。一艘小得可怜，但带有动力充足的马达与崭新渔网，在阳光下闪烁着漂亮耀眼的光泽的渔船。苏格兰说：“我知道你喜欢海，船长。”于是船长跳上船，像将军巡视站队的军队一样，身后站着他的卫兵，迎着海风，巡视他的海之王国。

　　“这感觉就像一千多年前，斯科特，”他缓慢地说，“我发现了你，你也发现了我……像你渡过海洋，至少在名义上对我伸出援手，我明知是假的，明知这艘渔船也走不了太远，但是忍不住信……”

　　苏格兰对他笑。新乌尔斯特远远地看着他们，把脸埋进自己风衣的领口，也止不住笑。他一边笑，一边咀嚼着被发现的，即使不是人类也可以享有的自由。
